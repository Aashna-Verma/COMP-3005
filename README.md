# Assignment 6 - Final project

## Materials 
- Install [Node](https://nodejs.org/en)  

## Instructions
1. Clone this repository
2. Go into the backend `cd backend `
3. Run `npm i`
4. Run `npm start`
5. Go into the frontend `cd ../frontend `
6. Run `npm i`
7. Run `npm start`
